## Dotfiles

My dotfiles.

## Manage with `rcm`

#### Install `rcm`

```sh
$ brew tap thoughtbot/formulae
$ brew install rcm
```

#### Clone the dotfiles repo

```sh
$ git clone git@github.com:anolson/dotfiles.git ~/.dotfiles
```

#### Install dotfiles

```sh
$ rcup
```
